# Smart Parking Availability App

An application that simulates parking lot availability by tracking which parking spots are occupied or available.  
The project focuses on backend logic in Java, unit testing, and interactive user interface.

## Features
- Manage parking spot availability 
- Update spot status when vehicles enter or exit
- Find the nearest available parking spot (lowest-numbered free spot)
- Interactive UI displaying available and occupied spots
- Live counters for total, available, and occupied spots
- Unit tests using JUnit to verify state transitions and availability logic

## How to Run

### Requirements
- Java (JDK 17 or newer)

### Steps
1. Clone the repository and open it in IntelliJ IDEA (or any Java IDE).
2. Build the project using Maven (IntelliJ handles this automatically).
3. Open the `ui/index.html` file in a web browser to view the parking availability UI.
4. Run the JUnit tests to verify the parking logic.
